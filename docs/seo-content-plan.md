# SEO Content Plan

Date: 2026-07-04

## Content Strategy

The SEO moat should be practical author operations, not generic "how to write a novel" content.

Each page should:

- solve a concrete launch/ops problem
- include a free tool, checklist, or template
- link to official/community sources
- avoid unsupported promises
- send users to the tool workspace or email hook

## Internal Link Pattern

- Every guide links to the relevant tool.
- Every tool links to 2-3 supporting guides.
- Launch planner is the central hub.
- Patreon calculator links to stats and author notes content.
- Shoutout tracker links to Discords and community rules.

## Email Hook

## Publishing Rule

No pages should be published until:

- user approves domain/deployment
- facts have source links
- community-sensitive copy is checked
- no AI/spam wording appears
